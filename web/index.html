Lenguaje sin ruido

Nodeon: sintaxis clara, JavaScript al final.

Minimalista como Python, compatible con el ecosistema JS/Node y pensado para escribir sin fricción: sin punto y coma, interpolación natural y tipado opcional.

Empezar rápido Ver sintaxis 
fn greet(name) {
  print("Hello {name}!")
}

fn scoreToGrade(score) {
  match score {
    case s when s >= 90 { "A" }
    case s when s >= 80 { "B" }
    default { "F" }
  }
}

result = scoreToGrade(87)
greet("Arcader")
print("Grade: {result}")

Getting Started

Requisitos: Node.js 18+ y npm.

  1. npm install — instala dependencias.
  2. node cli/bin.js build hello.no — compila a JavaScript.
  3. node cli/bin.js run hello.no — compila y ejecuta en memoria.

Tip: añade -min para minificar o --map para generar source maps.

// hello.no
fn main() {
  print("Hola {name}")
}

main()

Guarda el archivo, compila con node cli/bin.js build hello.no y ejecuta el .js resultante o usa run para hacerlo en un paso.

Playground minimal

Ahora puedes transpilar Nodeon a JavaScript directo en el navegador. Selecciona un snippet o pega el tuyo, pulsa "Transpilar" y mira el JS generado. La misma API que usa la CLI, sin backend.

Documentación básica

    

      

        
Variables

        
x = 42

Funciones

        
fn double(x) { x * 2 }
fn sum(a, b) = a + b
async fn fetch(url) { ... }

        
Cuerpo de una sola expresión ⇒ retorno implícito.

      

      

        
Control de flujo

        
if score >= 90 { print("A") }
else if score >= 80 { print("B") }
else { print("F") }

for i in 0..10 { print(i) }
while x > 0 { x = x - 1 }

      

      

        
Strings

        
"Hola {name}"  // interpolación
'sin {interpolación}'  // literal
`Hola ${name}`  // estilo JS
